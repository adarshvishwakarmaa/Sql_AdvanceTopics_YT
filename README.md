# SQL Ultimate Course

Welcome to the **SQL Ultimate Course** repository! 🚀
This course covers everything you need to master SQL.

---

## 🛠️ Tools & Resources

* **Datasets:** Course datasets (CSV files) available in the `datasets/` folder
* **SQL Server Express:** Lightweight database server
* **SQL Server Management Studio (SSMS):** Tool to manage and run SQL queries

---

## 📂 Repository Structure

```
sql-ultimate-course/
│
├── datasets/        # Raw datasets used in the course
├── docs/            # Documentation and notes
└── scripts/         # SQL scripts and queries
```
